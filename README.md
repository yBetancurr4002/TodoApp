# TodoApp

Project for academic purposes with HTML, CSS, Bootstrap and JS (+JQuery).
## Estructura del proyecto 

```css
todoapp/
├── index.html          # Archivo principal HTML
├── src/
│   ├── main.js         # Lógica JavaScript para agregar y eliminar tareas
│   └── style.css       # Hoja de estilos principal
└── README.md           # Documentación del proyecto
```

## Recomendaciones

Este proyecto es ideal para aprender los conceptos básicos de desarrollo web. Puedes mejorarlo agregando funcionalidades como: 

* Guardar Tareas en LocalStorage : Para que las tareas persistan incluso después de recargar la página.
* Editar Tareas : Permitir que los usuarios modifiquen las tareas existentes.
* Filtrar Tareas : Agregar opciones para mostrar tareas completadas o pendientes.
     

## Funcionalidades

1. Agregar una Tarea : 
    * Ingresa el nombre de la tarea en el campo de texto debajo del título.
    * Haz clic en el botón "Agregar Tarea" para añadirla a la lista.
2. Ver Tareas : 
    * Las tareas agregadas aparecerán en una lista debajo del botón.
3. Eliminar una Tarea : 
    * Haz clic en el botón "Eliminar" junto a la tarea que deseas quitar.
4. Validación : 
    * Si intentas agregar una tarea sin texto, se mostrará una alerta indicando que debes ingresar una tarea válida.
